Estamos en clases de AI y este repositorio tiene como objetivo demostrar cómo funciona GitHub

## YouTube Promo Site

Este repositorio ahora incluye una Landing Page en React (`youtube-promo`) diseñada para promocionar el canal de YouTube de Daniel Narváez.

Para ejecutar el proyecto localmente, asegúrate de tener Node.js instalado y ejecuta:

```bash
cd youtube-promo
npm install
npm run dev
```

Esto levantará el servidor de Vite en `http://localhost:5173` o `http://127.0.0.1:5173` configurado para ser accesible localmente o redencionando `host: true`.
